# Laser gate 
Gate for ROBO competition https://robovehicle.cz/

Measuring device for a robotics competition. The gate consists of 3D-printed parts and an electronics board with a Raspberry Pi.
The Raspberry Pi creates its own Wi-Fi network, on which a local webpage displaying lap times is accessible.

Many thanks to Ing. Michal Lekýr, PhD., for the complete mechanical design and PCB layout.
